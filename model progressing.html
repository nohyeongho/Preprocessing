  • A, B열: 모델 마스터 + 출처 1 모델 통합 (A=모델, B=Contents ID)
  • C, D열: 마이너 마킹 (C=PDF 여부, D=온라인 여부)
  • F, G, H열: 출처 2 데이터 매핑 (F=모델, G=Item ID, H=Contents ID)
  • J, K, L열: 출처 3 데이터 매핑 (J=모델, K=Item ID, L=Contents ID)
  • 절단 규칙: 5번째부터 나오는 하이픈(-) 이후 삭제 (중복 제거 포함)
